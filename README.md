# Owntone

This image is based on *Kafouche Alpine Linux*.

This image comes without the folllowing features:
- ChromeCast
- Spotify
- MPD
- Web User Interface
- ...

## Ports

| Port   | Description              |
|--------|--------------------------|
| `3689` | Default Port.            |

## Volumes

| Volume               | Description              |
|----------------------|--------------------------|
| `/var/cache/owntone` | Configuration Directory. |
| `/srv/music`         | Music and Media Files.   |
